# Leitura demonstrativa do grupo Emmanuel

Este material foi criado apenas para demonstracao do projeto.

Uma boa aula comeca com presenca tranquila, leitura atenta e disposicao para ouvir. Nem sempre sera possivel estudar muito, mas sempre e possivel retomar com sinceridade e constancia.

Para o encontro desta semana, o convite e simples:

- separar alguns minutos para leitura calma;
- anotar uma pergunta curta;
- observar uma atitude pratica que possa ser levada para o cotidiano.

Se surgir dificuldade, o grupo pode ajudar com respeito e fraternidade.
