# README da base de conhecimento

Este diretorio guarda a base de conhecimento do portal em formato Markdown.

Nesta etapa, a organizacao foi preparada por livro e por grupo:

- `data/knowledge/emmanuel/`
- `data/knowledge/a_caminho_da_luz/`

Tambem existe um arquivo `index.json` com metadados simples para busca, catalogacao e futura integracao.

## Objetivo da base

- apoiar respostas simples e revisaveis para alunos;
- dar material curto de apoio para professores;
- organizar resumos autorais sem expor PDFs;
- manter conteudo facil de revisar e atualizar.

## Regras de autoria

- cada arquivo deve ser curto e focado;
- nao incluir capitulo completo;
- nao incluir transcricao longa;
- usar linguagem simples, educativa e respeitosa;
- manter revisao humana como etapa obrigatoria.

## Estrutura sugerida por grupo

- `visao_geral.md`: panorama inicial do estudo do livro no grupo;
- `temas_centrais.md`: eixos de conversa e estudo;
- `palavras_chave.md`: termos, sinonimos e formas de pergunta;
- `duvidas_frequentes.md`: perguntas comuns de alunos e pontos para revisao do professor.

## Observacao de compatibilidade

Os arquivos antigos que continuam na raiz de `data/knowledge` foram preservados nesta etapa para nao exigir mudancas imediatas no backend ou no frontend.

Na proxima fase, a leitura automatica da base podera ser atualizada para considerar subpastas e usar o `index.json` como apoio.
